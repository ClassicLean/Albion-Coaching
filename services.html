Fix one mistake

Fix one mistake every session and build elite habits.

In a 60-min VOD review we find one repeat mistake, fix it, give a drill, and track one metric.

How a Coaching Session Works

0–5 min
  • Introduction
  • Discuss goals for the session
  • Identify what metric we will use
5–20 min
  • Watch selected moments from the VOD
20–35 min
  • Pick top 1 mistake
  • Explain the cause with a short rubric
35–50 min
  • Propose a fix and demonstrate it if possible
  • Guided drill with the student
50–58 min
  • Homework
  • Provide drill variants
58–60 min
  • Wrap up the session
  • Next session requirements

Do you offer feedback after the session?
